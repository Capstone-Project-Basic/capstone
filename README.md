
# 🐥PocketTeacher
### _2024. 06. 08 🥈은상 수상_


### 🎬 프로젝트 소개

_Kyonggi University Pre Capstone Design._


스마트폰 중독에 빠지는 우리의 어린이들... 

그것을 도와주는 어플리케이션이 등장했습니다.

친구들과 모여서 놀자! 핸드폰을 흔들어서 그것을 인증하자!

선생님은 활동적인 미션을 제공하고 도장을 지급하자!
